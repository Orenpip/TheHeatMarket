  COMPASS Project — Complete Summary                                                                                                                                                                                                        
                                                                                                                                                                                                                                              
  What COMPASS Is                                                                                                                                                                                                                           
                                                                                                                                                                                                                                              
  Student career exploration platform. High schoolers complete structured reflection sessions every 3 weeks → AI (Claude + Voyage AI) synthesizes responses into "signal profiles" (interest clusters, character signals, trajectory shifts,
  breadth score, compressed summary) → counselors see filtered dashboards. A pgvector-powered map surfaces personalized career/major recommendations.
  ---
  Architecture
  compass/
  ├── apps/web/          — Next.js 14 App Router (student + counselor + admin UI) → Vercel
  ├── apps/ai-service/   — FastAPI Python (synthesis, embeddings, prompts) → Railway
  ├── apps/mobile/       — React Native (untouched)
  ├── packages/db/       — Prisma 6 WASM + Supabase schema
  ├── packages/ui/       — Shared components
  ├── CONTRIBUTING.md    — Migration workflow docs
  Stack: Next.js 14, TypeScript, Tailwind, Prisma 6 (WASM engine), Supabase PostgreSQL + pgvector, Clerk auth, Upstash Redis + QStash, Anthropic claude-sonnet-4-6, Voyage AI (1024-dim embeddings)
  ---
  Database (14 models, 3 migrations applied)
  Key models: Student, Counselor, ReflectionSession, ReflectionTemplate, Activity, Reflection, SignalProfile, StudentPrivacySettings, MapNode (pgvector), Notification, NotificationPreferences.
  Key constraints added this session:
  - SignalProfile.@@unique([studentId, sessionId]) — prevents duplicate synthesis on QStash retry

  - StudentPrivacySettings.shareSummary Boolean @default(true)
