## Step 1: Supabase

**Go to:** [supabase.com](https://supabase.com) → New project

1. Pick a name (`compass-prod`), strong database password (save it), region closest to you
2. Wait ~2 min for provisioning

**Once created, do these in order:**

### Enable pgvector
Go to **SQL Editor** → run:
```sql
create extension if not exists vector;
```

### Get your connection strings
Go to **Settings → Database → Connection string**

You need two URLs:

**`DATABASE_URL`** — use the **Transaction pooler** (port 6543):
```
postgresql://postgres.[ref]:[password]@aws-0-[region].pooler.supabase.com:6543/postgres?pgbouncer=true
```

**`DIRECT_URL`** — use the **Session pooler** or direct connection (port 5432):
```
postgresql://postgres.[ref]:[password]@aws-0-[region].pooler.supabase.com:5432/postgres
```

> The `DATABASE_URL` goes through PgBouncer (required for serverless). The `DIRECT_URL` is used by Prisma Migrate to run schema changes directly.

Save both — you'll need them for Vercel and Railway env vars.
